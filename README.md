# Sectorial Topological Charge Detection

## Overview

This project presents a Deep Learning approach for detecting the **Topological Charge (TC)** of sectorial optical vortex beams. A synthetic dataset is generated for multiple TC values, and a neural network is trained to classify the corresponding topological charge with high accuracy.

The project automates the recognition of optical vortex beam patterns and can be useful in optical communication, beam characterization, and photonics research.

---

## Features

- Synthetic dataset generation
- Supports TC values from **-20 to +20**
- Automatic image preprocessing
- Deep Learning based classification
- High classification accuracy
- Easy-to-use Jupyter Notebook implementation

## Dataset

The dataset contains synthetic sectorial optical beam images generated for different Topological Charge (TC) values ranging from **-20 to +20**.

Each class represents a unique topological charge.

## Installation

Clone the repository

```bash
git clone https://github.com/pywithpranavi/Sectorial-TC-Detection.git
```

Move into the project directory

```bash
cd Sectorial-TC-Detection
```

Install dependencies

```bash
pip install -r requirements.txt
```

## Usage

Run the Jupyter Notebook:

```bash
jupyter notebook sectorial_tc_detection.ipynb
```

or execute it in Google Colab.
